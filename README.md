# 🚩 TryHackMe: TakeOver Walkthrough

## 📖 Overview
This repository contains a complete walkthrough of the **TakeOver** room on TryHackMe.  
This room focuses on understanding **Subdomain Takeover vulnerabilities**, a common yet critical security issue in web applications.

---

## 🎯 Objectives
- Understand what subdomain takeover is
- Learn how misconfigured DNS records create vulnerabilities
- Identify takeover opportunities
- Gain practical cybersecurity knowledge

---

## 🛠️ Tools Used
- Browser
- DNS analysis techniques
- Basic reconnaissance skills

---

## 🚀 Walkthrough

### Step 1: Understanding the Concept
Subdomain takeover happens when a subdomain points to an external service (like GitHub Pages, AWS, etc.) that is no longer in use.

---

### Step 2: Identifying Vulnerability
- Check DNS records
- Look for unused or dangling services
- Identify misconfigured subdomains

---

### Step 3: Exploitation Concept
If the external service is not claimed, an attacker can:
- Register the service
- Gain control of the subdomain
- Host malicious content

---

## 🧠 Key Learnings
- Importance of DNS hygiene
- Risks of unused subdomains
- Real-world impact of takeover attacks
